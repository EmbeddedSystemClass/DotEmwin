ValidTimeEventCode Class

Valid Time Event Code (VTEC). An event is defined as a specific combination of phenomenon and level of significance. Each VTEC event is given an Event Tracking Number, or ETN. There are two types of VTEC, the P-(Primary) VTEC, and an H-(Hydrologic) VTEC.
